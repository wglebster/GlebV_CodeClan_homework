What‚Äôs classification and how is it different to clustering?

It‚Äôs a machine learning model that is used to sort out different objects based on certain features.

  • Classification uses predifined classes in which objects are assigned.

  • Clustering identifies similarities between objects and groups the objects according to those common characteristics.

What‚Äôs Bayes Classifier good for?

It can help us determining probability of A happening, given that B has occured.

B is the evidence.

A is the hypothesis.

P(A|B) = (P(B|A)*P(A)) / P(B)

Two assumptions:

  • Predictors are independent.

  • All predictors have equal effect on the outcome. I.e. is is not more important how windy it is than how rainy it is.

Advantages of naive Bayes:

  • When assumption of feature independence holds true, the algorithm performs better than other models.

  • It requires relatively little amount of data, therefore it is quick to train.

  • It is (relatively) easy to implement.

Disadvantages of naive Bayes:

  • It‚Äôs main limitation is the assumption of independent predictors. It is almost impossible to achieve in real life.

  • If categorical variable has a category in test data set, which was not observed in training data set, then model will assign a 0 (zero) probability and will be unable to make a prediction. This is known as ‚ÄúZero Frequency‚Äù. This issue can be solved using smoothing.
